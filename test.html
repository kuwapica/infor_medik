Mental Health Test

Stres

1. Saya merasa tegang atau gugup hampir setiap hari.

2. Saya sering merasa sulit untuk berkonsentrasi.

3. Saya merasa sulit untuk tidur atau mendapatkan tidur yang nyenyak.

4. Saya merasa kelelahan meskipun tidak melakukan banyak aktivitas.

5. Saya sering merasa kewalahan dengan tanggung jawab sehari-hari.

Kecemasan

6. Saya merasa cemas tanpa alasan yang jelas.

7. Saya sering merasa takut atau khawatir akan hal buruk yang mungkin terjadi.

8. Saya merasa gelisah atau sulit untuk duduk diam.

9. Saya merasa sulit untuk menikmati hal-hal yang biasanya saya nikmati.

10. Saya sering mengalami ketakutan yang berlebihan terhadap hal tertentu.

Depresi

11. Saya merasa sedih atau hampa hampir setiap hari.

12. Saya merasa tidak berharga atau gagal.

13. Saya kehilangan minat pada aktivitas yang biasa saya nikmati.

14. Saya merasa sulit untuk memiliki energi atau motivasi.

15. Saya sering memiliki pikiran untuk menyerah atau merasa putus asa.

Keseimbangan Hidup

16. Saya memiliki waktu yang cukup untuk keluarga dan teman-teman.

17. Saya mampu menjaga keseimbangan antara pekerjaan atau belajar dan waktu pribadi.

18. Saya merasa puas dengan cara saya membagi waktu setiap hari.

19. Saya memiliki waktu luang yang cukup untuk menikmati hobi atau kegiatan lain.

20. Saya merasa tidak terlalu terbebani dengan tuntutan pekerjaan atau studi saya.
